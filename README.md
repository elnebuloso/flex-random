## Requirements

The following versions of PHP are supported by this version.

* PHP 5.4
* PHP 5.5
* PHP 5.6
* PHP 7.0
* HHVM

## Coding Standards

Flex follows the standards defined in the PSR-0, PSR-1, PSR-2 and PSR-4 documents.

## Installation / Usage

Via Composer

``` json
{
    "require": {
        "elnebuloso/flex-random": "~4.0"
    }
}
```
