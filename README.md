## Installation / Usage

Via Composer

``` json
{
    "require": {
        "elnebuloso/flex-random": "~2.0"
    }
}
```

## Requirements

The following versions of PHP are supported by this version.

* PHP 5.3
* PHP 5.4
* PHP 5.5
* PHP 5.6
* HHVM
